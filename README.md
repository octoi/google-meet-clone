### SETUP 👷‍♂️

> You need <a href="https://flutter.dev">Flutter</a> installed to run this project

Download This Project

```bash
$ git clone <CURRENT-PAGE-URL>
```

Install stuffs

```bash
$ flutter pub get
```

Run the project

```bash
$ flutter run
```
